# 拆分顶部队列独立信息层

## 修改原因

待投放水果队列放在原 HUD 区域内时，当前悬浮水果移动到顶部右侧会遮挡队列，造成视野冲突。单纯上移队列空间不足，因此需要从布局上拆出独立信息层。

## 主要修改

- 新增顶部独立信息层，集中显示标题、分数、最高分和 `QUEUE` 队列。
- 将水果生成线 / 死亡警戒线从 `120px` 下移到 `180px`。
- 当前悬浮水果被放置在顶部信息层下方，避免与待投放序列互相遮挡。
- 调整顶部队列图标尺寸和 q0 队首提示，让队列适配独立信息层高度。

执行 Python 编译检查、dummy 视频驱动 smoke test，并保存首帧截图检查顶部信息层和当前悬浮水果不再重叠。

## 备注

本次会改变手动游戏的生成线高度，等价于给顶部 UI 和当前水果预览留出更明确的空间。
