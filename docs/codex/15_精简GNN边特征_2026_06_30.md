# 精简 GNN 边特征

## 修改原因

在节点特征精简之后，对 `GraphBuilder` 中不同边关系的特征进行了第一轮审查。目标是减少边特征中的重复信息，让边主要表达“两个节点之间的关系”，避免把源节点或目标节点自身属性重复拷贝到边上。

## 主要修改

- `action ↔ board_fruit`
  - 删除 `distance`、`overlap_margin`、`drop_x_minus_fruit_x`。
  - 新增 `path_overlap_margin = radius_sum - horizontal_distance`，用于表达候选落点竖直路径和场上水果横向范围的重叠程度。
  - 该边现在更偏向描述投放路径、横向接触可能性和等级匹配。
- `board_fruit ↔ board_fruit`
  - 删除 `approaching_speed`。
  - 保留 `relative_vx`、`relative_vy`，避免保留过强的派生运动特征。
- `queue_fruit ↔ board_fruit`
  - 删除 `board_fruit_x`、`board_fruit_y`。
  - 场上水果位置由 `board_fruit_node` 自身提供，边只保留队列位置和等级匹配关系。
- `action ↔ queue_fruit`
  - 只保留 `queue_index`。
  - 删除 `action_index`、`is_current_queue_fruit`、`level_diff`、`abs_level_diff`、`same_level`，避免重复动作节点和队列节点已有信息。
- `queue_fruit ↔ queue_fruit`
  - 删除边上的 `queue_index`。
  - 保留 `order_gap`、`is_next_queue_fruit`、`level_diff`、`abs_level_diff`、`same_level`，用于表达队列顺序和相邻/同级规划关系。
- `board_fruit ↔ boundary`
  - 删除边上的 `is_left_wall`、`is_right_wall`、`is_floor`、`is_danger_line`。
  - 边界类型仍由 `boundary_node` 自身特征表达；边只保留 `distance_to_boundary` 和 `is_near_boundary`。
- 同步更新消融特征组，避免消融配置引用已删除的边特征。

边特征维度从早期的 `35` 降到 `26`。边界节点特征本身未被删除。

## 涉及文件

- `src/daxigua_rl/graph/schema.py`: 精简 `EDGE_FEATURE_NAMES`，将 `drop_x_minus_fruit_x` 替换为 `path_overlap_margin`。
- `src/daxigua_rl/graph/builder.py`: 调整各类边的特征生成逻辑，并补充边构建段落的说明注释。
- `src/daxigua_rl/graph/ablation.py`: 同步更新边特征消融组。

## 验证方式

- `python -m py_compile src/daxigua_rl/graph/schema.py src/daxigua_rl/graph/builder.py src/daxigua_rl/graph/ablation.py`
- 使用 `HeadlessGame` 构造投放后局面，调用 `GraphBuilder().build(...)` 验证图构建正常。
- 针对以下关系分别做 smoke test：
  - `action_to_board_fruit`
  - `board_fruit_to_board_fruit`
  - `queue_fruit_to_board_fruit`
  - `action_to_queue_fruit`
  - `queue_fruit_to_queue_fruit`
  - `board_fruit_to_boundary`
- 验证 `no_board_motion`、`no_queue_planning`、`no_action_board_relation` 等消融预设仍可正常应用。
- `git diff --check`

## 备注

- 本次只精简已有边特征，没有新增诱导连续合成的专门特征。
- 后续如果要支持“q0/q1 连续合成后匹配场上水果”的规划能力，可在 `queue_fruit ↔ queue_fruit`、`queue_fruit ↔ board_fruit` 或 `action ↔ board_fruit` 上单独设计链式合成提示特征。
