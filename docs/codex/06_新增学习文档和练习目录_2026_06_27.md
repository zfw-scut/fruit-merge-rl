# 新增学习文档和练习目录

## 修改原因

`docs/learning.md` 明确了本项目是一种基于真实项目、学习资料和 Agent 辅助的强化学习项目化学习流程。为了支撑后续学习规划、练习设计和学习者编码，需要将学习文档与练习代码分开管理。

## 主要修改

- 新增 `docs/learning/`，用于存放学习路线、阶段规划、练习说明、学习笔记和 Agent 协作规则。
- 新增 `docs/learning/ROADMAP.md`，作为第一版强化学习项目化学习路线。
- 新增 `docs/learning/RULES.md`，约束 Agent 生成学习内容和练习说明的方式。
- 新增 `practice/`，作为学习者练习代码和实验脚本空间。
- `practice/` 初始保持干净，仅使用 `.gitkeep` 保留目录，具体练习结构等需要时再创建。
- 更新项目文件索引，说明学习文档区和练习区的职责边界。

## 涉及文件

- `docs/learning/README.md`
- `docs/learning/RULES.md`
- `docs/learning/ROADMAP.md`
- `practice/.gitkeep`
- `docs/project_map/PROJECT_FILE_INDEX.md`

## 验证方式

已检查目录结构和 Markdown 文档生成情况。

## 备注

本次只新增学习规划和练习空间，不修改游戏接口，也不开始实现自动游玩/RL 代码。
