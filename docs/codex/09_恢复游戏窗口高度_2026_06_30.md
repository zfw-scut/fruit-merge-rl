# 恢复游戏窗口高度

## 修改原因

上一次将固定窗口高度调整为 `533px` 后，实际手动游戏场地过矮，影响游玩体验和水果堆叠观察。

## 主要修改

- 将固定窗口尺寸从 `400x533` 恢复为 `400x800`。
- 保留新增的顶部待投放水果队列 `QUEUE`。
- 保留窗口固定策略，不恢复拖动窗口边框缩放。

## 涉及文件

- `src/daxigua/config.py`
- `README.md`
- `docs/project_map/PROJECT_FILE_INDEX.md`

## 验证方式

执行 Python 编译检查和 dummy 视频驱动 smoke test，确认窗口尺寸恢复为 `400x800`，队列仍正常初始化。
