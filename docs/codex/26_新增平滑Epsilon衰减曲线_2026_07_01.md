# 新增平滑 Epsilon 衰减曲线

## 修改原因

旧训练入口只支持按环境步数线性衰减 epsilon。用户希望 epsilon 能按训练进度以更平滑的曲线下降，并大致满足：

```text
0%   -> 1.00
30%  -> 0.50
50%  -> 0.20
70%  -> 0.07
80%+ -> 0.05
```

## 主要改动

- 在 `src/daxigua_rl/scripts/train_dqn.py` 新增 `--epsilon-schedule` 参数。
- 默认 schedule 改为 `smooth`。
- 保留旧线性衰减方式，可通过 `--epsilon-schedule linear` 使用。
- 新增 smoothstep 分段插值曲线，按 `update_step / total_updates` 计算训练进度。
- 新增 `tests/test_epsilon_schedule.py`，固定 smooth 曲线锚点、单调性和旧 linear 行为。

## 设计说明

- warmup 阶段仍然使用 `epsilon=1.0` 进行随机探索。
- 正式训练阶段默认使用 smooth schedule。
- smooth schedule 内部使用归一化衰减比例，因此如果后续修改 `epsilon_start` 或 `epsilon_end`，曲线形状仍会保持相同趋势。

## 使用方式

默认不需要额外参数：

```bash
--epsilon-schedule smooth
```

恢复旧线性方式：

```bash
--epsilon-schedule linear \
--epsilon-decay-steps 50000
```
