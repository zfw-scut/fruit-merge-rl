# 删除旧 RL 相关内容

## 修改原因

旧训练脚本和旧环境封装已经不再作为当前方向继续维护，后续相关能力会重新设计。为了降低项目结构复杂度，先移除历史训练代码，让当前仓库聚焦游戏本体。

## 主要修改

- 删除旧环境封装和三套历史训练脚本。
- 删除 Paddle 版本训练依赖的旧 ResNet 网络文件。
- 更新 `README.md`，将项目说明改为当前游戏本体。
- 更新 `requirements.txt`，移除训练依赖，只保留当前游戏运行依赖。
- 更新项目文件索引，删除旧训练入口和旧环境说明。

## 涉及文件

- 删除 `State.py`
- 删除 `train_keras.py`
- 删除 `train_torch.py`
- 删除 `train_paddle.py`
- 删除 `resnet.py`
- 更新 `README.md`
- 更新 `requirements.txt`
- 更新 `docs/project_map/PROJECT_FILE_INDEX.md`
- 更新 `docs/codex/RULES.md`
