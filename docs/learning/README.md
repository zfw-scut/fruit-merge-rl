# Learning Docs

本目录用于存放强化学习项目化学习相关文档。

这里的内容服务于学习过程本身，包括学习路线、阶段规划、知识点解释、练习说明、阶段复盘和 Agent 协作规则。

## 目录说明

- `ROADMAP.md`: 当前学习路线总览。
- `RULES.md`: Agent 生成学习文档和练习说明时需要遵守的规则。
- `stages/`: 按阶段沉淀学习计划、阶段目标和阶段总结。
- `exercises/`: 练习题说明、任务要求和验收标准。
- `notes/`: 学习者或 Agent 沉淀的概念笔记、问题记录和复盘。

## 与其他目录的边界

- `docs/learning/`: 放学习规划和说明文档，不放正式项目代码。
- `practice/`: 放学习者练习代码、草稿和实验。
- `src/daxigua/`: 游戏本体代码。
- `src/daxigua_rl/`: 后续正式自动游玩/RL 模块。

学习练习中的代码先放在 `practice/`。只有当某段代码经过理解、验证并决定长期维护时，才考虑迁移到 `src/daxigua_rl/`。
